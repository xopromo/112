🧗 Поиск лучших соседних параметров

Начинает с выбранного результата и ищет лучшие варианты среди соседних параметров. Продолжает копать в перспективных направлениях — как горный восход по склону.
ИСТОЧНИК СТАРТОВЫХ ТОЧЕК
🎲 Multi-start HC: случайно расставляет N стартовых точек по всему пространству параметров и из каждой запускает Hill Climbing. Находит глобальный оптимум вместо локального — лучший результат из всех точек становится победителем. Эффективнее чем один старт в 5–10×.
⚡ Быстрый OOS-скан всех видимых → заполняет robScore ≥ N для выбора выше
ЦЕЛЕВАЯ МЕТРИКА
📊 TV метрика: каждый сосед тестируется дважды — на IS (70%) и на полных данных. Beam search ищет стратегии, которые не теряют эффективность на новых данных. ~2× медленнее P/DD.
⚡ В режиме Устойчивость: каждый сосед проверяется на всех 5 тестах (OOS★★★★★ → WF → Param → Noise → MC). Итераций авто-лимит 30 — каждая итерация ~2–5 сек. Beam search идёт в направлении максимальной устойчивости, не просто лучшей метрики. P/DD — тай-брейкер.
