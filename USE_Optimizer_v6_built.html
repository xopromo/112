ИСТОЧНИК СТАРТОВЫХ ТОЧЕК
🎲 Multi-start HC: случайно расставляет N стартовых точек по всему пространству параметров и из каждой запускает Hill Climbing. Находит глобальный оптимум вместо локального — лучший результат из всех точек становится победителем. Эффективнее чем один старт в 5–10×.
⚡ Быстрый OOS-скан всех видимых → заполняет robScore ≥ N для выбора выше
